概率论的概率表达整理

这是一份整理的、面向考研数学一的概率论与数理统计公式“大礼包”。我将用最朴实无华的语言，力求让你一看就懂，快速掌握核心计算方法。

我们将内容分为三个部分：
第一部分：概率论基础（事件的运算）
第二部分：随机变量及其分布（核心计算）
第三部分：数理统计（套路总结）

第一部分：概率论基础 (事件的运算)

这部分是地基，搞懂了事件之间的关系，后面的概率计算才能顺畅。把 A、B 看作两个集合或者两个圈。

  1. 加法公式 (求并集/“或”的概率)

    • 公式： P(A U B) = P(A) + P(B) - P(AB)
    • 大白话解释： 求两个圈合起来的面积，就是A圈面积 + B圈面积。但中间重叠的部分 (AB) 被加了两遍，所以要减掉一次。
    • 特殊情况： 如果 A 和 B 互不相容 (互斥)，意思是它俩没有重叠部分 (AB = ∅)，那么 P(A U B) = P(A) + P(B)。

  2. 减法公式

    • 公式： P(A - B) = P(A) - P(AB)
    • 大白话解释： A 发生但 B 不发生的概率。就是从 A 圈里，把 A 和 B 重叠的那一块挖掉。

  3. 对立事件 (逆事件)

    • 公式： P(Ā) = 1 - P(A)
    • 大白话解释： “A不发生”的概率，就是用总的概率 1 减去 “A发生”的概率。这个公式超级好用，有时候直接求 P(A) 很麻烦，算 P(Ā) 却很简单。

  4. 条件概率 (核心概念)

    • 公式： P(B|A) = P(AB) / P(A) (其中 P(A) > 0)
    • 大白话解释： 在 A已经发生 的这个前提下，B 发生的概率。可以理解为，我们的“全世界”从原来的样本空间缩小到了事件 A 的范围，然后看 B 在这个新世界里占多大比例。

  5. 乘法公式 (求交集/“且”的概率)

    • 公式： P(AB) = P(A) * P(B|A)
    • 大白话解释： A 和 B 同时发生 的概率。等于“A先发生”的概率，再乘以“在A发生的前提下B也发生”的概率。这个公式就是把条件概率公式变了个形，但常用于计算“接连发生”的事件。
    • 推广： P(ABC) = P(A) * P(B|A) * P(C|AB)

  6. 事件的独立性 (非常重要！)

    • 定义： A 的发生不影响 B 发生的概率，即 P(B|A) = P(B)。
    • 判定/计算公式： P(AB) = P(A) * P(B)
    • 大白话解释： 如果两个事件相互独立，那么它俩同时发生的概率就等于各自概率的乘积。比如，你抛硬币是正面，不影响我掷骰子是6点。
    • 推广： 如果 A, B, C 两两独立，并且 P(ABC) = P(A)P(B)P(C)，则 A, B, C 相互独立。
    • 有用的小结论： 如果 A, B 独立，那么 Ā 与 B，A 与 B̄，Ā 与 B̄ 也都相互独立。

  7. 全概率公式 (把复杂问题分解)

    • 公式： 假设 B₁, B₂, …, Bₙ 是样本空间的一个划分 (互不相容，且并集为全集)，则 P(A) = P(B₁)P(A|B₁) + P(B₂)P(A|B₂) + ... + P(Bₙ)P(A|Bₙ)
    • 大白话解释： “分情况讨论”。想求 A 的概率，可以把它分解成“在 B₁ 发生时 A 发生”、“在 B₂ 发生时 A 发生”…… 等多种情况，然后把每种情况的概率加起来。本质是“先分类，再汇总”。

  8. 贝叶斯公式 (知道结果，反推原因)

    • 公式： P(Bᵢ|A) = [ P(Bᵢ)P(A|Bᵢ) ] / [ Σ P(Bⱼ)P(A|Bⱼ) ] (分母就是全概率公式)
    • 大白话解释： 我们已经观察到了结果 A，现在想反过来推断，这个结果是由原因 Bᵢ 导致的概率有多大。比如，在“警报响了(A)”这个结果下，反推“是由于着火(B₁)导致”的概率。

第二部分：随机变量及其分布 (核心计算)

这部分是概率论的血肉，是考研计算的绝对主力。

一维随机变量

特征 离散型随机变量 (比如掷骰子) 连续型随机变量 (比如身高、时间)
描述工具 分布律 P(X = xᵢ) = pᵢ 概率密度函数 f(x)
核心性质 1. pᵢ ≥ 0
2. Σ pᵢ = 1 (所有可能取值的概率加起来是1)		 1. f(x) ≥ 0
2. ∫ f(x)dx = 1 (从-∞到+∞对密度函数积分为1)
求概率 P(a ≤ X ≤ b) = Σ pᵢ (在范围内的概率直接相加) P(a ≤ X ≤ b) = ∫ₐᵇ f(x)dx (概率是密度函数在区间的定积分/面积)
分布函数 F(x) F(x) = P(X ≤ x) = Σ pᵢ (所有 xᵢ ≤ x 的概率之和) F(x) = P(X ≤ x) = ∫₋∞ˣ f(t)dt (密度函数从-∞到x的积分)
数学期望 E(X) (平均值) E(X) = Σ xᵢpᵢ E(X) = ∫ xf(x)dx
方差 D(X) (离散程度) D(X) = E{[X - E(X)]²} = Σ [xᵢ - E(X)]²pᵢ D(X) = E{[X - E(X)]²} = ∫ [x - E(X)]²f(x)dx
方差简便公式 D(X) = E(X²) - [E(X)]² (这个公式在计算时最常用！) D(X) = E(X²) - [E(X)]²
期望/方差性质 1. E(aX + b) = aE(X) + b
2. D(aX + b) = a²D(X)
3. D(c) = 0 (常数的方差是0)		 (同左)

常见分布 (必须背熟！)

分布名称 故事/背景 分布律/密度函数 f(x) E(X) D(X)
0-1分布 做1次实验，成功或失败 P(X=k) = pᵏ(1-p)¹⁻ᵏ, k=0,1 p p(1-p)
二项分布 B(n,p) 做n次独立重复实验，成功k次 P(X=k) = C(n,k) pᵏ(1-p)ⁿ⁻ᵏ np np(1-p)
泊松分布 P(λ) 单位时间/面积内事件发生次数 P(X=k) = (λᵏ / k!) * e⁻λ λ λ
均匀分布 U(a,b) 在区间(a,b)内取值，机会均等 f(x) = 1/(b-a) for a<x<b, else 0 (a+b)/2 (b-a)²/12
指数分布 E(λ) 等待事件首次发生的时间 f(x) = λe⁻λx for x>0, else 0 1/λ 1/λ²
正态分布 N(μ,σ²) 自然界中最常见的分布 f(x) = (1/(σ√2π)) * e^[-(x-μ)²/(2σ²)] (这个函数不用背，但要认识) μ σ²

正态分布的重要性质：

  • X ~ N(μ, σ²)，则 Z = (X - μ) / σ ~ N(0, 1) (标准化)。
  • P(X ≤ a) = Φ((a - μ) / σ)，其中 Φ(x) 是标准正态分布的分布函数，查表可得。
  • Φ(-x) = 1 - Φ(x) (对称性)。

二维随机变量

特征 离散型 (X, Y) 连续型 (X, Y)
联合分布律/密度 P(X=xᵢ, Y=yⱼ) = pᵢⱼ f(x, y)
边缘分布 P(X=xᵢ) = Σⱼ pᵢⱼ (对y求和)
P(Y=yⱼ) = Σᵢ pᵢⱼ (对x求和)		 f_X(x) = ∫ f(x,y)dy (对y积分)
f_Y(y) = ∫ f(x,y)dx (对x积分)
条件分布 `P(Y=yⱼ X=xᵢ) = pᵢⱼ / P(X=xᵢ)`
独立性判断 pᵢⱼ = P(X=xᵢ) * P(Y=yⱼ) 对所有 i,j 成立 f(x,y) = f_X(x) * f_Y(y)

协方差和相关系数 (衡量两个变量的关系)

  • 协方差： Cov(X, Y) = E{[X - E(X)][Y - E(Y)]} = E(XY) - E(X)E(Y)
  • 相关系数： ρ_{XY} = Cov(X, Y) / [√D(X) * √D(Y)]
  • 性质：
    • X, Y 独立 ⇒ Cov(X, Y) = 0 ⇒ ρ_{XY} = 0 (反之不成立！ 线性无关不代表独立)。
    • D(aX ± bY) = a²D(X) + b²D(Y) ± 2abCov(X, Y) (这个公式超级重要！)
    • 如果 X, Y 独立，则 D(aX ± bY) = a²D(X) + b²D(Y)。
    • |ρ_{XY}| ≤ 1。ρ 越接近1，正相关性越强；越接近-1，负相关性越强；越接近0，线性相关性越弱。

重要结论 (考研常用)

  • 若 (X, Y) 服从二维正态分布，则 ρ_{XY} = 0 等价于 X, Y 相互独立。
  • E(X+Y) = E(X) + E(Y) (永远成立)
  • 若 X, Y 独立，则 E(XY) = E(X)E(Y)。

大数定律和中心极限定理 (理论支柱)

  • 切比雪夫不等式： P{|X - E(X)| ≥ ε} ≤ D(X) / ε² (知道就行，考计算少)
  • 中心极限定理： 设 X₁, X₂, ..., Xₙ 独立同分布，E(Xᵢ)=μ, D(Xᵢ)=σ²。当 n 足够大时，ΣXᵢ 近似服从 N(nμ, nσ²)，样本均值 X̄ 近似服从 N(μ, σ²/n)。
    • 大白话： 很多独立同分布的随机变量加在一起，不管它们原来是什么分布，最后的结果都像正态分布。这是正态分布如此重要的原因。

第三部分：数理统计 (套路总结)

这部分就是概率论知识的应用，方法和套路非常固定。

三大抽样分布 (统计学的基石)

  1. 卡方分布 (χ²)

    • 构造： X₁, ..., Xₙ 独立同服从 N(0,1)，则 Y = ΣXᵢ² ~ χ²(n) (n个标准正态的平方和)。
    • 性质： χ₁² + χ₂² ~ χ²(n₁+n₂) (可加性)

  2. t 分布

    • 构造： X ~ N(0,1), Y ~ χ²(n), 且X,Y独立，则 T = X / √(Y/n) ~ t(n)。
    • 用途： 在样本方差 S² 未知时，用于检验均值。

  3. F 分布

    • 构造： U ~ χ²(n₁), V ~ χ²(n₂), 且U,V独立，则 F = (U/n₁) / (V/n₂) ~ F(n₁, n₂)。
    • 用途： 用于检验两个正态总体的方差是否相等。

参数估计

目标：通过样本 (X₁, ..., Xₙ) 来估计总体中的未知参数 (如 μ, σ², p 等)。

  1. 矩估计法 (MME)

    • 思路： 用样本矩估计总体矩。
    • 步骤：
      1. 计算总体的 k 阶原点矩 E(Xᵏ) (通常算一阶E(X)和二阶E(X²)就够了)，结果会是参数的表达式。
      2. 计算样本的 k 阶原点矩 Aₖ = (1/n) Σ Xᵢᵏ。
      3. 令 E(Xᵏ) = Aₖ，联立方程组，解出参数的估计值。
    • 大白话： 让样本的平均值等于总体的期望，让样本的平方的平均值等于总体的平方的期望，然后解方程。

  2. 最大似然估计法 (MLE)

    • 思路： 什么样的参数，能让我们手头这组样本出现的可能性最大？
    • 步骤：
      1. 写出似然函数 L(θ) = Π f(xᵢ; θ) (离散型是 Π P(X=xᵢ; θ))。
      2. 取对数：ln L(θ) = Σ ln f(xᵢ; θ)。
      3. 求导：令 d(ln L)/dθ = 0。
      4. 解出 θ 的估计值 θ̂。
    • 大白话： 构造一个函数，这个函数代表了“当前这组样本出现的概率”，然后求这个函数的最大值点。
